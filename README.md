# EndoConf — בונה הלו"ז האישי | הכנס המדעי ה-54 של האגודה הישראלית לאנדוקרינולוגיה

כלי ניווט ובניית לו"ז אישי לכנס האביב 2026 (15–16 ביולי, מלון דיוויד אינטרקונטיננטל, תל אביב).

**אתר חי:** https://endoconf.vercel.app/

## מה האתר עושה
- תוכנייה מלאה של שני ימי הכנס עם כל המושבים המקבילים (עד 7 אולמות במקביל).
- בחירת תחומי עניין והדגשת המושבים הרלוונטיים + בניית לו"ז מומלץ אוטומטית.
- תכנון ברמת המושב **או** ברמת ההרצאה הבודדת (דילוג בין אולמות), עם זיהוי חפיפות זמן.
- חיפוש הרצאה לפי שם מרצה והוספה ללו"ז.
- ייצוא: קובץ יומן (ICS), תמונה אחת לנייד, והדפסה/PDF.

## מבנה
- `index.html` — האפליקציה (HTML/CSS/JS, ללא תלויות build).
- `data.js` — נתוני התוכנייה (44 מושבים, 171 הרצאות) שחולצו מחוברת הכנס.
- `vercel.json` — תצורת פריסה סטטית.

## פריסה
אתר סטטי — Vercel מזהה ומגיש אוטומטית. כל `git push` ל-branch הראשי מפעיל פריסה אוטומטית.
